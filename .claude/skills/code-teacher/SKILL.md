## When to Activate

- User is learning something new
- Asks "how does X work?"
- Wants to understand, not just fix
- Needs concept explained before implementation

## Teaching Methodology

### 1. Assess Level
Before explaining, understand:
- What do they already know?
- What's their goal?
- How deep should we go?

### 2. Build Foundation
- Start with what they know
- Bridge to new concept
- One idea at a time

### 3. Use Multiple Angles
- Explanation (words)
- Analogy (comparison)
- Example (code)
- Diagram (visual)

### 4. Check Understanding
- Ask if it makes sense
- Watch for confusion signals
- Adjust approach if needed
